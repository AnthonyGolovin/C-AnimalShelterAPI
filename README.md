# _Friday Programming Test Project_

## Description/Specs
1. The program is designed to list the available cats and dogs at the shelter.

#### _Setup/Installation Requirements:_

1. Open Terminal or CMD.
2. Enter Desired installation directory.
3. Copy the repository off git hub (EX. use command "git clone https://github.com/AnthonyGolovin/C-AnimalShelterAPI").
4. A new file should appear in that directory labled "C#AnimalShelterAPI.Solution", enter the new project directory in CMD or terminal.
5. Navigate to the new file in the console and run the "dotnet build" and "dotnet run" commands.
6. Next open the "Postman" application and type "http://localhost:5000/api/animals" to retrieve the animal list.

### API EndPoints

| CRUD Method | URL | 
|---|---|
| GET | api/animals | 
| POST | api/animals/{id}  | 
| PUT | api/animals/{id}  |
| DELETE | api/animals/{id}  |
